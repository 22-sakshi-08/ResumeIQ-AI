# Contributing Guidelines

Thank you for your interest in contributing to **ResumeIQ AI**!

## How to Contribute

1. **Fork the Repository**: Create a personal copy of the repository on GitHub.
2. **Clone Locally**:
   ```bash
   git clone https://github.com/username/AmazonMl.git
   cd AmazonMl
   ```
3. **Set Up Environments**:
   * Refer to the `README.md` for backend virtual environment and frontend node modules setup instructions.
4. **Create a Branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
5. **Ensure Code Quality**:
   * Run code formatting checks (Ruff, Black).
   * Ensure all tests pass:
     ```bash
     pytest tests/
     cd src/frontend && npm run test
     ```
6. **Submit a Pull Request**: Submit your pull request explaining the changes, and wait for review.
